# Client APIs

### What is the Client API?

A client API is a way for users to interact with the ICON blockchain. The [API Endpoints](/develop-on-icon-chain/networks#rpc-endpoints) host this API. Accessing their API can either be done directly via Remote Procedure Call (RPC) or through a libary written in a specific programming language.

### Available APIs

#### RPC-based

* [JSON-RPC API](/develop-on-icon-chain/client-apis/json-rpc-api)

#### Language-based

* [Javascript SDK](/develop-on-icon-chain/client-apis/javascript-sdk)
* [Python SDK](/develop-on-icon-chain/client-apis/python-sdk/)
